# pyekfmm
Pyekfmm: A Python Package for 3D Fast-Marching-Based Travel-Time Calculation and Its Applications in Seismology

<img src='https://github.com/aaspip/gallery/blob/main/pyekfmm/test_pyekfmm_fig3.png' alt='Slicing' width=960/>

## Getting started 

This package only requires the pyekfmm package

```
pip install pyekfmm
```

All scripts in examples reproduce the results in the following paper 

## Reference
    Chen Y., Chen, Y.F., Fomel, S., Savvaidis, A., Saad, O.M., Oboue, Y.A.S.I. (2023). A python package for 3D fast-marching-based traveltime calculation and its applications in seismology, Seismological Research Letters, 94, 2050-2059.
